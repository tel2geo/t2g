WTHR - API

WTHR - API permit to display the 5 days forecast for a selected city and country. In order to try the API (application programming interface) SIMPLE BAN, you should say the following sentence.

Example

Quelque chose de plus intéressant...

Dites "temps" puis ajouter le nom de la ville ainsi que le pays dans votre phrase (ex: "temps Paris France").

Pour naviguer sur le site : Dites "accueil", "prénoms", "météo", ou "biorythme", "géolocalisation" ou "recharger la page".

More

The WTHR api is available on gitlab there.

WTHR links

  1. ip-api.com - the JSON API that permit to resolve IP to precise coordinates link.

  2. OpenWeatherMap.org - The API that permit to asynchonously display the 5 days weather's forecast link.

  3. Nominatim - OpenStreetMap.org - The fantastic API that permit to resolve city-country name to coordinates link.
